# Programmierung 2 – Tutorium (WDS125 & WDS225) 🚀

Willkommen im offiziellen Repository für das Tutorium zum Modul **Programmierung 2**. Dieses Repo wird von **Jonas** und **Darian** gepflegt und dient dazu, den Studierenden der Kurse **WDS125** und **WDS225** zusätzliche Lernmaterialien und Übungsmöglichkeiten bereitzustellen.

## 📖 Über dieses Repository

In diesem Repository findet ihr begleitende Materialien, um die Vorlesungsinhalte zu vertiefen und praktisch anzuwenden. Unser Ziel ist es, euch bestmöglich auf die kommende Klausur vorzubereiten.

Das Repository ist modular aufgebaut, sodass ihr schnell die passenden Themen findet. Jedes Thema umfasst in der Regel zwei Teile:

1.  **Übungsaufgaben:** Hier findet ihr Aufgaben mit Problemstellungen zu den relevanten Themen (z.B. Datenstrukturen, Steams, etc.).
2.  **Lösungen:** Zu jeder Übung stellen wir Lösungen bereit, damit ihr euren Code vergleichen und nachvollziehen könnt.

---

## 🛠 Nutzung des Repos

Um die Inhalte lokal auf eurem Rechner zu nutzen, könnt ihr das Repository einfach klonen:

```bash
git clone https://github.com/Kingofstrix/TutoriumProgrammierung2.git
```

oder natürlich das Repo ganz normal herunterladen.

Viel Spaß beim Lernen!
